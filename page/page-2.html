    Github Sites:

  • Adopt A Plant

    AdoptAPlant is a web-based service that allows users to adopt plants for a monthly subscription. Users then being able to watch the plants via live streaming video, monitor environmental conditions, and toggle the lights or water the plant via the press of a button.


  • Train Time

    This project uses trains to give practice using firebase (server-side hosting) and moment.js to calculate time automatically and populate them to a table.


  • Pokedex 2.0

    This is a play on the classic Pokedex from the Pokemon games. It features a search field where the user can type in the name of a pokemon. It would access an API that would display a variety of features of that particular Pokemon, such as health, attack, defense, as well as an image of the Pokemon.


  • Friend Finder

    Want to date but hate actually going out? Here's Friend-Finder. Think if Facebook, Tinder, and OKCupid had a baby. It's the app no one asked for.


  • MongoNews

    A news scraping service designed to streamline finding and reading articles from trusted news sources.
